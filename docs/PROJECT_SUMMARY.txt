QUICK START FOR MAC
================================================================================

1. INSTALL TOOLS (One-time, 10 minutes):
   brew install awscli ansible terraform
   aws configure

2. CHECK PREREQUISITES:
   cd Online-Courses/scripts
   ./check-prerequisites.sh

3. DEPLOY WEBSITE (3 minutes):
   cd ../ansible
   ansible-playbook playbooks/ansible-only-deploy.yml

4. SETUP MONITORING (5 minutes):
   cd ../nagios
   ./setup-nagios-mac.sh
   ./update-monitoring-url.sh YOUR_WEBSITE_URL

5. CLEANUP WHEN DONE:
   cd ../ansible
   ansible-playbook playbooks/ansible-cleanup.yml

================================================================================
TROUBLESHOOTING
================================================================================

AWS Credentials Error:
  aws configure
  aws sts get-caller-identity

Ansible Not Found:
  brew install ansible

Website Not Accessible:
  Wait 30 seconds for S3 propagation

Nagios Not Starting:
  brew services restart nagios

For detailed troubleshooting, see DEVOPS_SETUP_GUIDE.md
